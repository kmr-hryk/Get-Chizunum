Get-Chizunum
====

## これは何

地籍フォーマット2000の筆属性情報ファイル(*.atr)から所在、地番、地図番号を抜出します。

## 使い方

第一引数に地籍フォーマット2000の筆情報ファイル(*.atr)を指定してします。  
相対パスでも大丈夫な気がします。

```
PS1> Get-Chizunum.ps1 B:\tmp\10201_2014_1.atr
```

とすると`B:\tmp\10201_2014_1.csv`にデータが出力されます。

データ形式は下記の様に、第一カラムに所在、第二カラムに地番、第三カラム以降存在する限り地図番号が1カラム1地図番号という形で格納されます。  
`%所在%,%地番%,%地図番号%,%地図番号%...`
